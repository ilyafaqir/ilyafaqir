## Installation (rapide)
```bash
# cloner
git clone https://github.com/TON_USERNAME/TON_PROJET.git
cd TON_PROJET

# créer venv (python)
python -m venv .venv
source .venv/bin/activate   # mac/linux
.venv\Scripts\activate      # windows

pip install -r requirements.txt

# lancer l'exemple
python run_example.py
